## Build

You require the following to build Nuklei:

* Latest stable [Oracle JDK 8](http://www.oracle.com/technetwork/java/)
* 3.0.4 or later of [Maven](http://maven.apache.org/)

To build and install to local maven repository.

    $ mvn clean install
